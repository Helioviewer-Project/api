# API Docs

This directory contains the Helioviewer API documentation source.

## Structure

- `source/` — Sphinx RST documentation source files, organized by API group
- `openapi.yaml` — OpenAPI 3.0.3 specification for the full API (hand-written, canonical)
- `requirements.txt` — Python deps for building the Sphinx docs

## OpenAPI spec

`openapi.yaml` was generated by reading the RST docs and covers all 25 endpoints across 7 tags:

| Tag | Endpoints |
|-----|-----------|
| JPEG2000 | downloadImage, getJP2Header, getJP2Image, getJPX, getJPXClosestToMidPoint, getStatus |
| Movies | queueMovie, postMovie, reQueueMovie, getMovieStatus, downloadMovie, playMovie |
| Screenshots | takeScreenshot, postScreenshot, downloadScreenshot, getEclipseImage |
| Official Clients | getClosestImage, getDataSources, getTile, shortenURL |
| Solar Features & Events | events |
| YouTube | checkYouTubeAuth, getYouTubeAuth, uploadMovieToYouTube, getUserVideos |
| Website | getNewsFeed |

The spec is maintained by hand — there is no automated RST-to-OpenAPI converter. When RST docs change, update `openapi.yaml` manually.
